# AI 기반 키오스크 이용 보조 서비스

> 오픈소스프로그래밍(01분반) | 2026학년도 1학기 팀 프로젝트

---

## 프로젝트 개요

카페, 패스트푸드점 등 일상 곳곳에 보급된 무인 키오스크는 편리함과 동시에 디지털 취약계층(노년층, 장애인, 외국인)에게 높은 진입 장벽이 되고 있습니다.  
실제로 60세 이상 노인의 디지털 기기 활용률은 47%에 불과하며, 키오스크를 이용하지 못해 방문 자체를 포기하는 사례가 늘고 있습니다.

본 프로젝트는 스마트폰 카메라로 키오스크 화면을 실시간 인식하고, 음성으로 단계별 안내를 제공해 누구나 키오스크를 독립적으로 이용할 수 있도록 돕는 AI 보조 서비스입니다.

---

## 핵심 기능

| 모듈 | 기능 |
|------|------|
| **Kiosk Screen Recognition** | 카메라 입력 → UI 영역(버튼·메뉴·텍스트) 실시간 탐지 (YOLOv5) |
| **OCR** | 탐지된 버튼 내 텍스트 추출 및 의미 변환 |
| **Voice Guidance (TTS)** | 현재 진행 단계 파악 → "오른쪽 아래 결제 버튼을 눌러주세요" 같은 음성 안내 생성 |

```

├── dataset/
│   ├── classes.txt           # 데이터 클래스 목록

├── vision/                   # 비전 인식 모듈
│   └── yolo_detector.py      # YOLOv5 ONNX 객체 탐지

│   ├── kiosk_run_demo.py         # ★ 데모 실행 파일 (저장된 데모 사진으로 동작 시연)

│   ├── kiosk_guide_1.py          # 음성 안내 모듈 본체 (멘트 생성 + gTTS)
│   └── README.md
```
